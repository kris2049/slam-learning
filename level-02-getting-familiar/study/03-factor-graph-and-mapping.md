5|## 3.1 因子图 (Factor Graph)
6|
7|### 3.1.1 概念
8|
9|**因子图 = 变量节点 + 因子节点**，将 SLAM 问题建模为**二分图**。
10|
11|- **变量节点** $\bigcirc$：待优化的量（相机位姿 $x_i$、路标 $l_j$）
12|- **因子节点** $\blacksquare$：约束关系（观测、里程计、先验）
13|
14|**因子** $\phi(\text{variables})$ = 势函数 = $\exp(-\text{error}^2)$
15|
16|$$\text{total probability} \propto \prod_k \phi_k(\text{neighbors}_k)$$
17|
18|取负对数 → 最小二乘问题：
19|
20|$$\min \sum_k \|\text{error}_k\|^2$$
21|
22|### 3.1.2 因子类型
23|
24|| 因子 | 连接 | 误差函数 |
25||------|------|----------|
26|| **先验因子** | $x_1$ | $\|x_1 - x_1^0\|$ — 固定初始位姿 |
27|| **里程计因子** | $x_i, x_{i+1}$ | $\|\log(T_{i,i+1}^{-1} T_i^{-1} T_{i+1})\|$ |
28|| **观测因子** | $x_i, l_j$ | $\|\pi(T_i, l_j) - z_{ij}\|$ — 重投影误差 |
29|| **回环因子** | $x_a, x_b$ | $\|\log(T_{ab}^{-1} T_a^{-1} T_b)\|$ — 同里程计但 $|a-b|$ 大 |
30|| **GPS因子** | $x_i$ | $\|t_i - t_i^{\text{GPS}}\|$ — 绝对位置约束 |
31|
32|### 3.1.3 优化库
33|
34|| 库 | 语言 | 特点 |
35||----|------|------|
36|| **g2o** | C++ | ORB-SLAM 使用，图优化专用 |
37|| **GTSAM** | C++ | Georgia Tech，因子图理论完善 |
38|| **Ceres** | C++ | Google，通用非线性优化 |
39|| **gtsam-python** | Python | 学习用 |
40|| **miniSAM** | Python | 轻量级因子图 |

42|### 3.1.4 增量推理 (iSAM2)
43|
44|SLAM 中不断有新观测。每次重解全量问题太慢。
45|
46|**iSAM2** (GTSAM)：增量平滑与建图
47|- 只重解受新因子影响的子树
48|- 用贝叶斯树 (Bayes Tree) 维护消元顺序
49|- 实现实时因子图求解

53|## 3.2 外点剔除 (Outlier Rejection)
54|
55|特征匹配不可避免地有错误匹配 → 需要过滤。
56|
57|### RANSAC (Random Sample Consensus)
58|
59|**核心思想**：随机采样最小点集估计模型，看多少人同意。
60|
61|```
62|1. 随机选 s 个点 (最小点集: 求F用8点, 求H用4点)
63|2. 用这 s 个点估计模型
64|3. 计算所有点到模型的距离
65|4. 距离 < 阈值的点 = 内点 (一致集)
66|5. 如果内点数 > 历史最优，更新最佳模型
67|6. 重复 N 次
68|```
69|
70|**需要多少次迭代？**
71|
72|$$N = \frac{\log(1-p)}{\log(1-w^s)}$$
73|
74|- $p$：期望成功率（通常 0.99）
75|- $w$：内点比例
76|- $s$：最小点集大小
77|
78|如果 50% 是外点 ($w=0.5$)，八点法 ($s=8$)：
79|$$N \approx 1177  iterations$$
80|
81|→ 这就是为什么有 RANSAC 变体来加速。
82|
83|### PROSAC (Progressive SAC)
84|
85|先用质量好的匹配（低描述子距离），逐步加入差的。
86|
87|### M-Estimator
88|
89|对残差加权，减小大残差的权重（等价于鲁棒核函数）。

93|## 3.3 建图 (Mapping)
94|
95|SLAM 的「M」— 不仅要知道「我在哪」，还要知道「世界什么样」。
96|
97|### 3.3.1 稀疏地图 — 点云
98|
99|**ORB-SLAM 输出**：稀疏的 3D 点云（每个点 = 一个 ORB 特征）。
100|
101|- 轻量，适合定位
102|- 不可直接用于导航/避障
103|
104|### 3.3.2 占据栅格地图 (Occupancy Grid Map)
105|
106|将空间离散化为 2D/3D 栅格，每个格记录「被占概率」。
107|
108|$$P(\text{occupied} \mid z_{1:t})$$
109|
110|**OctoMap**：八叉树结构，高效存储 3D 占据信息。
111|- Loose（不占）→ Partial（部分）→ Full（全占）
112|- 层次化：大块空区域用粗节点，细节用细节点
113|
114|### 3.3.3 TSDF (Truncated Signed Distance Function)
115|
116|**KinectFusion 的核心**。
117|
118|每个体素存储到最近表面的**截断带符号距离**：
119|
120|$$D(\mathbf{x}) = \begin{cases} \min(1, d/\mu) & d > 0 \\ \max(-1, d/\mu) & d \le 0 \end{cases}$$
121|
122|- $d$：体素到表面的实际距离
123|- $\mu$：截断距离（如 5cm）
124|- 表面 = TSDF 零交叉面
125|
126|**体积融合**（多帧累积）：
127|
128|$$D_{fused} = \frac{W_{old} D_{old} + W_{new} D_{new}}{W_{old} + W_{new}}$$
129|
130|$$W_{fused} = W_{old} + W_{new}$$
131|
132|加权平均 → 噪声逐渐消除，表面越来越清晰。
133|
134|### 3.3.4 Surfel (面元)
135|
136|**ElasticFusion 的表示**。
137|
138|Surfel = 带有半径的圆形面片，包含：
139|- 位置、法向量、颜色
140|- 半径（覆盖的圆盘大小）
141|- 置信度
142|
143|优势：非刚体变形（回环时弹性拉伸 Surfel 地图）
144|

147|## 3.4 传感器模型
148|
149|### IMU (惯性测量单元)
150|
151|包含加速度计 + 陀螺仪：
152|
153|$$\mathbf{a}_m = \mathbf{a}_{true} + \mathbf{b}_a + \mathbf{n}_a + \mathbf{g}$$
154|$$\boldsymbol{\omega}_m = \boldsymbol{\omega}_{true} + \mathbf{b}_g + \mathbf{n}_g$$
155|
156|- $\mathbf{b}$：**偏置 (bias)** — 即使静止也有非零读数，随时间缓慢变化
157|- $\mathbf{n}$：白噪声
158|- $\mathbf{g}$：重力加速度
159|
160|**IMU 噪声模型**：
161|- 白噪声 → 高斯
162|- 偏置 → 随机游走（布朗运动）
163|- 用 **Allan 方差** 分析 IMU 噪声特性
164|
165|### LiDAR
166|
167|发射激光脉冲 → 测量飞行时间 → 得到3D点云。
168|
169|- 机械旋转式 (Velodyne)：360° 扫描
170|- 固态 (MEMS/OPA/Flash)：无运动部件
171|- 与视觉互补：不受光照影响，但雨天退化
172|
173|### 多传感器标定
174|
175|**相机-IMU 标定**（Kalibr工具箱）：
176|- 时间对齐（时间戳偏移 $t_d$）
177|- 空间对齐（$T_{ci}$：IMU→相机的外参）
178|
179|**相机-LiDAR 标定**：
180|- 检查板+边缘提取
181|- 最小化点到平面的距离

185|## 3.5 评测指标
186|
187|### ATE (Absolute Trajectory Error)
188|
189|全局轨迹精度。先做 Sim(3) 对齐（消除全局坐标系差异）：
190|
191|$$ATE = \sqrt{\frac{1}{N}\sum_{i=1}^N \|\mathbf{t}_i^{\text{est}} - \mathbf{t}_i^{\text{gt}}\|^2}$$
192|
193|### RPE (Relative Pose Error)
194|
195|局部漂移。比较固定时间差 $\Delta$ 的相对运动：
196|
197|$$RPE = \sqrt{\frac{1}{N-\Delta}\sum_{i=1}^{N-\Delta} \|\log((T_i^{-1}T_{i+\Delta})^{-1} T_i^{\text{gt}-1}T_{i+\Delta}^{\text{gt}})\|^2}$$
